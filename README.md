# Electric Vehicle Range Estimation Using Machine Learning

## Overview

This project focuses on predicting the driving range of electric vehicles (EVs) using machine learning techniques. It addresses the limitations of traditional range estimation methods by incorporating multiple dynamic factors such as environmental conditions, driving behavior, and battery characteristics.

**Current Status:**  
This project is under active research and development and has not yet been deployed.

---

## Objective

1. Develop regression models to estimate EV driving range with improved accuracy  
2. Compare the performance of multiple machine learning algorithms  
3. Identify key factors influencing EV range  
4. Lay the foundation for a reliable real-time prediction system  

---

## Problem Statement

Conventional EV range estimation methods rely heavily on State of Charge (SoC) and often produce inaccurate predictions, with errors ranging between 20–30%.

This project aims to overcome these limitations by addressing:

- Non-linear relationships between variables such as speed and energy consumption  
- The impact of driving behavior and traffic conditions  
- Environmental influences such as temperature  
- Battery degradation over time
